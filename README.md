# Análise de Redes Sociais: Preferências Musicais

Este projeto utiliza Python e a teoria de Grafos para analisar as conexões entre estudantes e seus gostos musicais. Através de dados coletados via formulário, o algoritmo gera matrizes, projeta redes de similaridade e calcula métricas topológicas para entender a dinâmica do grupo.

## Tecnologias Utilizadas

* [Python 3.x](https://www.python.org/)
* **Pandas:** Manipulação de dados e criação de matrizes.
* **NetworkX:** Criação, manipulação e cálculo de métricas de grafos.
* **Matplotlib:** Visualização gráfica das redes.
* **NumPy:** Operações de álgebra linear.

## Pré-requisitos

Certifique-se de ter o Python instalado. Em seguida, instale as dependências necessárias executando o comando abaixo no terminal:

```bash
pip install pandas networkx matplotlib numpy openpyxl
```

Execute o script principal:

```Bash
python main.py
